# My Utils

实用工具集合

## 项目结构

```
my-utils/
├── frontend/          # 前端项目
│   ├── public/       # 静态资源
│   ├── src/          # 源代码
│   │   ├── components/  # React组件
│   │   └── ...
│   ├── package.json  # 依赖配置
│   └── ...
└── README.md         # 项目说明
```

## 功能列表

### 番茄时钟

一个简单的番茄工作法计时器，帮助你保持专注。

特性：
- 可自定义时长的计时器
- 优雅的界面设计
- 滴答声效果
- 完成提醒
- 声音开关控制

## 开发

### 前端开发

```bash
cd frontend
npm install
npm start
```

浏览器会自动打开 http://localhost:3000
